A modern, elegant multi-page fashion storefront built entirely with pure HTML & CSS —
no frameworks, no dependencies, just crafted code and clean design.

---

## 🗂️ Pages

| Page | File | Description |
|------|------|-------------|
| 🏠 Home | `index.html` | Hero banner, category grid, featured products & promo |
| 🛍️ Shop | `shop-products.html` | Filter sidebar, 9-product grid & pagination |
| 🛒 Cart | `shop-cart.html` | Cart items, quantity controls & order summary |

---

## 🎨 Design Highlights

- **Palette** — Warm cream `#F5F0E8` · Charcoal `#1A1A1A` · Terracotta `#C4622D` · Gold `#C9A84C`
- **Typography** — Playfair Display (headings) + DM Sans (body)
- **Aesthetic** — Editorial luxury meets minimal warmth

---

## ⚡ Features

- 🎠 Animated scrolling marquee announcement bar
- 🖱️ Hover-reveal "Add to Cart" buttons on product cards
- 🎛️ Filter sidebar with category, price range, color swatches & size
- 🛒 Cart with live quantity +/− controls
- 📦 4-step checkout progress indicator
- 🏷️ Sale / New / Hot product badge labels
- 🔗 Fully linked navigation across all pages
- 🔒 Secure checkout badge & payment icons
